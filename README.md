# Shoppers Mart Inc.

Welcome to the Shoppers Mart Inc. repository! This repository houses the frontend of the Shoppers Mart e-commerce platform, built using the reactJs with Vite. The primary focus of this implementation is the seamless integration of the frontend with the Stripe backend implimented using Django-Oscar, enabling efficient online payment processing.

## Setup Instructions

1. Check Node Version:
   ```bash
   node -v
   ```

2. Check npm Version:
   ```bash
   npm -v
   ```

3. Installing npm dependencies
    ```bash
   npm install
   ```

4. Start Development server
    ```bash
   npm start
   ```

4. Access the Application
`http://localhost:<port>`

## Note: Check the active port for the development server where the development server is running.
